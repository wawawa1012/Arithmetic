## 1. 八皇后问题 (N-Queens)

**📝 题干描述：**
在一个 8x8 的国际象棋棋盘上放置 8 个皇后，要求它们互不攻击。即任意两个皇后不能处于同一行、同一列或同一斜线上。求出所有的合法摆放方案。

**💡 核心思路 (DFS + 回溯)：**
* **按行推进：** 每一行必定只有一个皇后，因此可以通过递归函数 `trace_back(int r)` 逐行向下放置，避免了复杂的同行冲突检查。
* **空间判定 (AABB思想延伸)：** 使用 `vector<point> queens` 动态记录已放置皇后的坐标。判断对角线冲突时，无需建立复杂的二维映射，直接利用数学绝对值关系：`abs(当前列 - 已存列) == abs(当前行 - 已存行)` 即可精准拦截。
* **状态回溯：** 无论下一行是否找到解，当前行的 `for` 循环在尝试下一个列位置前，必须执行 `queens.pop_back()` 弹出上一次的尝试，恢复棋盘状态（读档）。

## 2. 迷宫寻路 (Maze Pathfinding)

**📝 题干描述：**
给定一个 5x5 的二维数组表示迷宫（`0` 代表空地，`1` 代表墙壁）。要求玩家从起点 `(0, 0)` 出发，只能上、下、左、右移动，寻找一条到达终点 `(4, 4)` 的通关路径，并输出该路径的坐标序列。

**💡 核心思路 (DFS + 状态标记)：**
* **边界与障碍拦截：** 递归函数的起手式先处理异常，遇到数组越界、或者踩到墙壁 `1` 直接 `return false`。
* **防止死循环 (撒面包屑)：** 在 2D 地图上寻路，极易发生“左右横跳”的无限递归。因此在踩上安全格子后，立刻将其值修改为 `2`（标记为已访问），断绝后代递归重走回头路的可能。
* **多米诺骨牌式返回：** 使用 `if (dfs(下一步)) return true;` 进行四周探测。只要最深处摸到了终点 `(4, 4)`，这个 `true` 就会沿着调用栈一路向上传递，瞬间终止其他无意义的方向探索。
* **清理现场：** 如果四个方向全是死胡同，在 `return false` 撤退前，必须把当前格子的 `2` 恢复为 `0`，并从路径 `vector` 中弹出坐标。

## 3. 单词搜索 (Word Search)

**📝 题干描述：**
给定一个 m x n 的二维字符网格 `board` 和一个字符串单词 `target`。如果 `target` 存在于网格中，返回 `true`。单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中“相邻”单元格是那些水平相邻或垂直相邻的单元格。同一个单元格内的字母在一个单词中不允许被重复使用。

**💡 核心思路 (全局搜索 + 索引追踪)：**
* **双层遍历寻找起点：** 目标单词的第一个字母可能在网格的任意位置，因此必须在外部使用双层 `for` 循环遍历整个 `board`，一旦内部的 `dfs` 命中目标，立刻跳出。
* **严格的执行顺序拦截：** 1. 检查坐标越界。
    2. 检查当前字母是否匹配 `target.at(index)`。
    3. 检查是否为最后一个字母 `index == target.size() - 1`（如果是，直接保存并宣告胜利，避免触发越界报错）。
* **完美现场保护：** 由于题目限制“同一个字母不能重复使用”，在进入下一层递归前，使用 `char temp = board[r][c]` 记住当前字母，并将棋盘上的该位置设为 `' '`（空格）作为访问标记。探索失败回溯时，再执行 `board[r][c] = temp` 将字母精准归还。
